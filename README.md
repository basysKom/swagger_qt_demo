# Building and Running

## Prerequisits

- Qt >=5.12
- Node.js >=12
- openapi-generator >=5.1.0

## OpenAPI

Take a look at our [Blog Post](https://blog.basyskom.com/2021/qml-qt-quick-and-swagger-openapi-tutorial) to see how to generate the code.

## ERP Server

    cd ERP\ Server
    npm install
    npm run start

## Machine Terminal

    cd MachineTerminal
    qmake
    make
    ./MachineTerminal
